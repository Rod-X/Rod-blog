# 柯里化
+ 当一个函数有多个参数的时候先传递一部分参数调用它，如何这部分参数永远不变，
+ 然后返回一个新的函数接受剩余的参数，返回结果

## ladash中的柯里化函数
+ _.curry(fn)
    - 直到函数收集完所有的参数再执行函数
```javascript
function myCurry(fn){
  return function curriendFn(...args){
    if(args.length<func.length){
      return function (){
        return curriendFn(...args.concat(Array.from(arguments)))
      }
    }
    return func(...args)
  }
}
```

## 实践总结
+ 柯里化可以帮我们缓存一些参数
+ 让函数更灵活，让函数的颗粒度更小
+ 把多元函数转换成一元函数，可以组合使用产生强大的功能
